# GNEC Hackathon 2026

Full featured eco system that aligns with UN 2030 SDG3 (health and well being) that provides access to healthcare for everyone in the world for free using many options depending on person's cases

## Apps

1. Platform
    - Donors system
    - Connected hospitals
    - in-person donors/workers to visit poor countries and find people with no phone or internet to access hospitals or healthcare at any time so they provide them with physical Lora like button 
    - Dashboard system
2. Hospital emergency system (Go + HTML + CSS + JS + P2P -> when Electricty is available ELSE Lora buttons `C Lang`)
3. P2P communication between devices when no internet in city or from anyone in the city to get healthcare access
4. Lora Button + Sound Button (Wowki)

## Stack

1. HTML
2. CSS
3. JavaScript (minimal)
4. Go 
5. SQLite (database)
6. C
